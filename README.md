# 🌐 Personal Portfolio Website

This is my personal developer portfolio built using **React, Vite, and Tailwind CSS**.  
The portfolio showcases my projects, technical skills, and contact information in a modern and responsive user interface.

The website is designed to highlight my experience as a **Full Stack Developer** and demonstrate my ability to build clean, responsive, and interactive web applications.

## 📌 Features

- Modern responsive portfolio design
- Smooth scrolling navigation
- Animated UI using Framer Motion
- Interactive hero section with typing animation
- Projects showcase section
- Skills and technology section
- Contact section
- Fully responsive for desktop, tablet, and mobile devices

---

## 🛠️ Tech Stack

**Frontend**

- React.js
- Vite
- Tailwind CSS

**Libraries**

- Framer Motion (animations)
- React Icons
- React Scroll
- React Type Animation
